# LeetCode22
#### 题目描述
给出 n 代表生成括号的对数，请你写出一个函数，使其能够生成所有可能的并且有效的括号组合。
#### 输入、输出示例
Example 1:
> Input:    n = 3

> Output:   [
              "((()))",
              "(()())",
              "(())()",
              "()(())",
              "()()()"
            ]

#### 解题思路
这道题用递归的思路求解。如果当前左括号的个数小于n，则可以放置一个左括号。
如果右括号的个数小于左括号的个数，则可以放置一个右括号。
递归结束的条件是：某一个可能的组合，字符串长度肯定为2n。
